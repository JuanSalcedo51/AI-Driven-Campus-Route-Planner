# AI-Driven-Campus-Route-Planner
Which is the best parking lot to reach a building in the least amount of time and effort, where time, capacity and proximity to the destination building are the most important factors?

This is an easy-to-medium level problem in the implementation of classical AI techniques. A student arriving on campus by car must find an available parking space that is as close as possible to the building where their first class is held. Instead of always going to the same parking lot and walking for several minutes, given the number of parking lots available to students, this agent seeks to reduce travel time by both driving and walking, and reach the destination building quickly and efficiently, taking into account the probability of parking availability (possible spots), entrance, destination building, and the time of day.
